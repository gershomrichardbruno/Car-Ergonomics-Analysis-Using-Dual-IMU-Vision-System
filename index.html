Car Ergonomics Analysis Using Dual IMU-Vision System

Real-time biomechanical analysis of car seat entry/exit movements using integrated IMU sensor network and computer vision technology.

Explore Methodology

About The Research

Real-Time Biomechanical Analysis

This research project develops a novel dual-methodology approach that combines multi-sensor IMU networks with computer vision validation to analyze car seat entry/exit movements.

With the average person spending over 100 minutes daily in vehicles, and millions of car entry/exit cycles performed annually, understanding the biomechanical demands of these transitions is essential for preventing injury and optimizing automotive design.

Traditional ergonomic assessments rely on subjective questionnaires or expensive laboratory-based motion capture systems that cannot be easily deployed in real-world settings. Our approach offers a cost-effective, portable solution for quantitative biomechanical analysis.

Research Methodology

IMU Sensor Network

Ten strategically placed BMI323 inertial measurement units providing comprehensive 3D kinematic data with real-time analysis and <100ms latency.

Computer Vision

YOLOv8-pose estimation for visual validation and cross-system accuracy verification with real-time pose detection at >30 FPS.

Human Subjects Validation

Testing with 15+ subjects across diverse demographics and multiple vehicle types (sedan, SUV, sports car) for comprehensive data collection.

Research Objectives

System Development

Develop 10-sensor IMU network using BMI323 sensors and ESP32 microcontroller with real-time data acquisition at 100Hz sampling rate.

Computer Vision Integration

Implement YOLOv8-pose estimation for visual validation with One Euro filtering for smooth coordinate tracking.

Biomechanical Analysis

Develop comprehensive analysis algorithms for movement assessment including joint angle calculations and motion phase detection.

Key Objectives & Outcomes

Primary Research Goals

Develop and validate an integrated IMU-vision system for real-time biomechanical analysis

Achieve ±5° joint angle accuracy validated against reference measurements

Create a comprehensive database of movement patterns across diverse populations

Generate evidence-based recommendations for automotive ergonomic design

Establish normative databases for comparison and injury risk assessment

Expected Impact

Improved vehicle accessibility for elderly and mobility-impaired users

Injury prevention through early detection of harmful movement patterns

Data-driven seat design optimization for automotive industry

Portable assessment capability for ergonomics consulting

Evidence-based ergonomic safety standards
